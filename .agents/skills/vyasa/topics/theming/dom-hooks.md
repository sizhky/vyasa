# DOM Hooks

Prefer stable hook classes over positional selectors.

Shell hooks include:

- `.vyasa-navbar-shell`, `.vyasa-navbar-card`
- `.vyasa-content-grid`, `.vyasa-main-shell`
- `.vyasa-sidebar`, `.vyasa-sidebar-card`
- `.vyasa-sidebar-toggle`, `.vyasa-sidebar-body`
- `.vyasa-posts-sidebar`, `.vyasa-toc-sidebar`
- `.vyasa-mobile-panel`, `.vyasa-mobile-panel-header`, `.vyasa-mobile-panel-body`
- `.vyasa-footer-shell`, `.vyasa-footer-card`

Do not teach selectors like `#site-navbar > *`, `#site-footer > *`, or `#posts-sidebar details > summary`.
