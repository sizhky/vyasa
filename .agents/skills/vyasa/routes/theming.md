Rules:

- Use explicit Vyasa hook classes, not positional selectors.
- Theme implementation lives in `vyasa/theme_extensions/`.
- `vyasa/extensions_builtin/themes/` is compatibility surface.
- For code blocks, use `--vyasa-code-*` CSS variables.
